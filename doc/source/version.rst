=============
appio version
=============

Displays the versions of the appio dlls.

***************
Usage
***************

.. code-block:: bash
  
   appio --version 

******************
Options 
******************
+------------------------+------------------------+
|Options                 |Description             |
+========================+========================+
| \\--version            |prints the dll-versions |
+------------------------+------------------------+

********************
